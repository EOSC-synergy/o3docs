The example above is how to use a Jupyter notebook to retrieve data points of either :doc:`tco3_zm plot <../ozonescience/tco3_zm>` or 
:doc:`tco3_return plot <../ozonescience/tco3_return>` in the JSON format and plot them.
The corresponding Jupyter notebook can be downloaded `here <https://git.scc.kit.edu/synergy.o3as/o3docs/-/raw/master/user/jupyter/o3api_plot_tco3.ipynb?inline=false>`__ .


.. tip::
   In order to run a Jupyter notebook one may use either `Jupyter Hub <https://notebook.learn.eosc-synergy.eu/hub/>`__ of `EOSC-Synergy <https://www.eosc-synergy.eu/>`__ 
   or `Colab <https://colab.research.google.com/>`__ free service from Google.
